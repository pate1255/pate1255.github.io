Heading tag
We can use the h1 h1 tag for the most important heading of the page. For the next level of heading we have h2/h2, h3/h3, h5, and h6/h6. The browser will style each heading differently, stating with the biggest size for h1 then h2 and so on.
Paragraph
For adding a paragraph of text we can use the tag. The browser will add a space before and after the content of the paragraph
